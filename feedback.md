# Project Review

## Project

**Name:** Dungeon Core

**One Sentence Pitch:** Starting with your core room, build a massive dungeon to challange adventuers.


# 1. First Impression
Spend 10-20 minutes using the project without making changes.
- dungeon should not start open, give the player a chance to build first
- the UI still doesn't feel clear, like we likley dont need the bottom section at all
- placing minions in rooms is not clear, it just increments a count, ideally it would be icons
- whats happening isn't clear, when adventuers are in and fight in the dungeon
- minons auto evolve, I actually dont want this to happen, we should unlock additional minons and we can swap them out if we want too, but not essential
- to many adventuer deaths should raise some level of warning that the dungeon is to dangerous and needs to be destroyed
- the initial race select screen isn't formatted

# 2. Story
Would I keep reading?
- this should be emergent style gameplay, but a tutorial should show us how to place buildings, create traps and place mobs
